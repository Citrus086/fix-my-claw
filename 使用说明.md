你可以通过以下几种方式启动 fix-my-claw：

## 方式一：CLI 命令行（推荐开发和服务器使用）

```bash
# 1. 确保已安装
cd /Users/mima0000/.openclaw/fix-my-claw
source .venv/bin/activate

# 2. 一键启动（初始化配置 + 开始监控）
fix-my-claw up

# 或者分步操作：
fix-my-claw init          # 初始化配置文件
fix-my-claw start         # 启用监控
fix-my-claw monitor       # 启动监控循环（前台运行）
```

**常用命令：**
```bash
fix-my-claw status        # 查看状态
fix-my-claw check         # 单次健康检查
fix-my-claw repair        # 手动触发修复
fix-my-claw stop          # 暂停监控
```

## 方式二：GUI 菜单栏应用（推荐桌面用户）

```bash
cd /Users/mima0000/.openclaw/fix-my-claw/gui

# 编译并运行
swift build
swift run

# 或者编译成可执行文件后运行
swift build -c release
.build/release/fix-my-claw-gui
```

GUI 会在菜单栏显示图标，可以：
- 查看实时状态（🟢健康/🔴异常）
- 手动触发检查/修复
- 安装/管理后台服务
- 接收 AI 修复确认弹窗

## 方式三：后台服务（推荐生产环境）

**macOS (launchd)：**
```bash
# 安装并启动服务
./deploy/launchd/install.sh --fix-my-claw-bin "$(command -v fix-my-claw)"

# 管理服务
fix-my-claw service status
fix-my-claw service start
fix-my-claw service stop
```

**Linux (systemd)：**
```bash
# 安装服务
sudo ./deploy/systemd/install.sh --fix-my-claw-bin "$(command -v fix-my-claw)"
sudo systemctl enable --now fix-my-claw.service
```
